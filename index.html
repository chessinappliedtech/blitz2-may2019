Регламент

Турнир №2 по блицу. Проводим с 20-го мая по 14-июня включительно (длительность ещё поправим в зависимости от количества участников).
Играем по круговой схеме с контролем времени 5 минут + 3 секунды добавление на ход.
Турнир двухраундовый, каждый играет с соперником две партии.
Конкретные дата и время игр не фиксированы, играем по готовности, но не затягиваем.
Коэффициенты: количество побед, Neustadtl (Sonneborn-Berger), Koya.
Если коэффициентов недостаточно для распределения мест, то играем стыковые матчи:
две партии с контролем 5+3, затем армагеддон (белые 5+3, чёрные 4+3, ничья трактуется в пользу чёрных), если игроков с одинаковыми коэффициентами больше чем два, придумаем схему позже.
